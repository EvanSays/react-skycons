react-skycons

skycons as React components

Demo

Installation

Install via npm

npm install --save react-skycons

Quick Example

var Skycons = require('react-skycons');
var Demo = React.createClass({

  render: function() {
    return (
      <Skycons color='white' icon='PARTLY_CLOUDY_DAY' autoplay={false}/>
    );
  }
});

Build

npm run build
